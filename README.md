# Wordly Dictionary Single Page Application

# Description
Wordly is a Single Page Application (SPA) dictionary tool. Users can search for any English word and instantly view its pronunciation, definitions, examples, and synonyms. Users can also save words to a Favorites.

# Features
- Word search
- Definitions and parts of speech
- Pronunciation text
- Audio pronunciation
- Example sentences
- Synonyms
- Favorites 
- Error handling (empty input, word not found, network failure)
- Light/dark theme toggle

# Technologies Used
- HTML
- Css
- JavaScript
- [Free Dictionary API](https://dictionaryapi.dev/)


# How to Run the Project
1. Clone or download the repository.
2. Open the `wordly` project folder.
3. Open `index.html` in a browser, or run it with the VS Code "Live Server" extension.
4. Enter a word in the search field and click "Search".

# API Information
`https://api.dictionaryapi.dev/api/v2/entries/en/{word}`
- The app retrieves meanings, pronunciation, audio, examples, synonyms, and source links when they are available in the API response.

# Usage
1. Enter a word in the search box.
2. Click "Search" button.
3. View the definition, part of speech, and pronunciation.
4. Click the speaker icon to play the audio pronunciation.
5. Click "Save" to add the word to Favorites, or click a saved word to search it again.
